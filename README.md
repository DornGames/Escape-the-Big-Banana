## 🎯 About the Game

**Escape the Big Banana** is an endless 3D survival game built with Unity. You are placed in a sprawling city map where five spawn points continuously generate giant bananas. These bananas will hunt you down relentlessly — your only defense is perfect movement and clever parkour. There are no levels, no checkpoints — just you, the city, and an ever-growing banana threat. The game ends when you get caught. Then you can quit or start a new round.

## 🕹️ Gameplay Features

| Feature | Description |
| :--- | :--- |
| 🏙️ **Open 3D City** | A fully explorable urban environment with streets, buildings, and obstacles |
| 🍌 **5 Spawn Points** | Bananas appear from fixed locations around the map |
| 🔍 **Homing AI** | Each banana actively tracks and chases the player |
| ⏱️ **Endless Mode** | No time limit — survive as long as you can |
| 💀 **Permadeath** | Game ends when a banana catches you |
| 🔄 **Restart Option** | Quit or start a new round after game over |

## 🗺️ Game Mechanics

| Mechanic | How It Works |
| :--- | :--- |

| 🎯 **Banana AI** | Bananas detect the player within a radius and begin chasing; speed increases gradually |

## 🎯 关于游戏

**《逃离大香蕉》** 是一款使用 Unity 开发的无限生存3D游戏。你将置身于一个庞大的城市地图中，地图上分布着五个生成点，每个生成点会定时生成大香蕉。这些大香蕉会主动寻找并追捕你——你唯一的防御手段就是完美的走位和巧妙的跑酷技巧。游戏没有关卡，没有检查点——只有你、城市，以及不断增长的香蕉威胁。游戏在你被抓住时结束。之后你可以退出或开始新一轮。
